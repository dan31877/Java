READ ME 

This project is a Full Stack Java Application using Spring MVC, JPA (EclipseLink), MariaDB, JSP, Servlet, HTML, CSS, and JavaScript. Further instructions on how to set up the project and troubleshooting can be found in the READ ME file in the Documents folder along with other documentation required for this assignment. This project is the culmination of a customized 12-week Full Stack Java Developer intensive bootcamp provided by Platform by Per Scholas in association with TEKSystems. 

This project is called the Deal Finder application. It's a web application that allows users to select a product and search which stores have the project and where it is the least expensive. This application is accessible for Registered Users along with Guest Users. Registered Users are also able to access their transaction history and update their profile information.
